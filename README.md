# Computer-Vision
This project comprises a comprehensive implementation of advanced Computer Vision pipelines, moving from fundamental signal processing to complex probabilistic modeling and optimization.

This repository features advanced Computer Vision implementations using Python 3.12, OpenCV 4.11, and NumPy 2.3.3. Core modules include "from-scratch" development of Canny edge detection, Hough transforms for circle detection, and Chamfer distance transforms. Geometric vision tasks involve checkerboard-based camera calibration, SIFT-based image stitching, and homography estimation for panoramic mosaics. Probabilistic state estimation is addressed through Extended Kalman Filters for non-linear unicycle models, while shape analysis is performed using Statistical Shape Models built via Generalized Procrustes Analysis and Probabilistic PCA. Optimization techniques include interactive object segmentation via Graph Cuts and Alpha-Expansion, along with energy function minimization for automated building extraction from high-resolution UAV imagery. Signal processing foundations cover Parseval’s theorem, magnitude-phase analysis in Fourier transforms, and stereo correspondence via Normalized Cross-Correlation. The framework further implements adaptive background subtraction using Gaussian Mixture Models for real-time foreground detection and person counting in video streams.
